About Monitor control

Welcome to the Monitor control web app! This web application allows you to run Python code directly from the browser, execute predefined functions, and interact with the system in a variety of ways.

Features:

  • Run Python code from a simple text input box.
  • Execute predefined functions to interact with the system, such as shutting down or rebooting the computer.
  • Easy-to-use interface with a modern design.

    • How it Works:

    This app uses Flask as the backend framework to execute Python code dynamically and PyAutoGUI for some of the predefined functions such as closing windows or shutting down the computer.

    Tips:

    The Python script has imported the following libraries: pyautogui, sys, os, io, win32api, win32con, win32gui, and win32process.
